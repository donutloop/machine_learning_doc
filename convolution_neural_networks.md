## Example of Layer
![convolution_network_layer_example](images/convolution_network_layer_example.png)

If layer l is a convolution layer:

* f[l] = fliter size
* p[l] = padding
* s[l] = stride
* nc[l] = number of filters 
* Input: n * n * nc
* output n * n * nc
* Each filter is: f[l] * f[l] *nc
